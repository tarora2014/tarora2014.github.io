Classes

When, Why, and How to use Classes

19th April, 2015

Classes in programming are program templates that can be used to create objects. They are useful in providing you the state of the object and ways to change the objects.

For example, imagine you are writing a program where you are describing different modes of transportation. You want to know things like the name of the transportation, how many people can it carry and if that mode travels on air, land or sea. In this example your class would be vehicles and your objects would be things like Car, Bus, Train, Boat, and Air-plains.

The names inside the initialize method are called instance variables which means every method inside the class has access to those variables. The book_ticket and pick_location are called instance methods which means that every time you create a new object with the vehicles class you will have access to those 2 methods.

The way to create a object for the Vehicles class is "car = Vehicles(car, 4, land)". The object in this case is a car, it can hold 4 people and it goes over land. If you wanted to call methods that are available to all things inside the Vehicles class you would just do "car.book_ticket(1)" or "car.pick_location("San Francisco")".
